# Reverse Geocode Widget for Geofield

A [Geofield] widget that provides a reverse geocode widget.

## Features:

- Pick location on GMap
- Reverse geocoding from coordinates to addressfield
- Supports single or multi-value Geofields (cardinality).

## Installation tips:

- Choose Reverse Geocode Widget on your Geofield field.

## Requires:

- [Geofield] (2.x)

[GeoPHP]: http://drupal.org/project/geophp
[Geofield]: http://drupal.org/project/geofield
